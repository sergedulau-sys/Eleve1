# Élevé — AI-First Luxury Experience Discovery

An AI concierge-powered platform that learns what you love and curates extraordinary experiences.

## How It Works
1. The AI concierge asks you 4 quick questions about your vibe, destination, budget, and group
2. Based on your answers, it curates a personalized selection of luxury experiences
3. Browse your recommendations, view details, and book

## Deploy to Vercel
```bash
cd eleve-app-v2
git init
git add .
git commit -m "v2 - AI concierge flow"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/eleve-app.git
git push -u origin main --force
```
Then go to vercel.com, import the repo, and deploy.

## Local Development
```bash
npm install
npm run dev
```
